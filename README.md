##### Installation:  

`git clone git://github.com/vicrucann/dotvim.git ~/.vim`

##### Create symlinks:  

`ln -s ~/.vim/vimrc ~/.vimrc`  
`ln -s ~/.vim/gvimrc ~/.gvimrc`  
`ln -s ~/.vim/zshrc ~/.zshrc`  

##### Switch to the `~/.vim` directory, and fetch submodules:

`cd ~/.vim`  
`git submodule init`  
`git submodule update`
